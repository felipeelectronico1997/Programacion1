Introduccion a My SQL

Estimado(a) estudiante, en esta seccion se tiene una lectura acerca de la introduccion a My SQL para la creacion de bases de datos y desarrollo de paginas web. Es una herramienta muy interesante y muy potente, tiene muchas aplicaciones y no es complicado a la hora de realizar los programas que se requieren para una actividad o trabajo especifico. Es una vision general, se dan temas y conceptos basicos pero importantes para aprender acerca de este tema. Tiene tambien un poco del lenguaje PHP visto anteriormente.
